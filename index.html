Mis proyectos

Contador de Vehículos en Tiempo Real

En uno de mis primeros proyectos en Python, desarrollé un programa que utiliza la biblioteca OpenCV y otras herramientas para contar vehículos en tiempo real. El código captura la pantalla a intervalos regulares, detecta vehículos en la imagen mediante la biblioteca cvlib y realiza un seguimiento de los mismos.

El programa utiliza diversas variables y estructuras de datos para llevar un registro del número de vehículos detectados, así como de sus posiciones. Cuando un vehículo es identificado, se verifica si ya fue contado anteriormente para evitar duplicaciones. Además, se implementa un mecanismo para identificar si un vehículo está en movimiento o parado, mejorando así la precisión del conteo.

La ventaja clave de este programa radica en su capacidad para ahorrar tiempo a las empresas al proporcionar un método automatizado y eficiente para contar vehículos en un entorno en tiempo real. La detección automática de vehículos elimina la necesidad de intervención manual, lo que no solo acelera el proceso, sino que también reduce posibles errores humanos.

Este proyecto demuestra mi habilidad para aplicar conceptos de programación en Python, trabajar con bibliotecas externas como OpenCV y cvlib, y abordar problemas prácticos como el conteo de vehículos en un entorno empresarial.

App Web en Node.js y mongoDB

Este proyecto, desarrollado en Node.js con MongoDB, consiste en una aplicación web para la gestión de documentación relacionada con la Ley Orgánica de Protección de Datos (LOPD). Es una solución completa que mejora la eficiencia en la gestión de documentos y fortalece la comunicación entre personal médico y pacientes.

La parte del servidor, definida en el archivo server.js, utiliza Express para la creación de un servidor web. La comunicación con la base de datos MongoDB se realiza a través de las rutas definidas en el archivo routes.js.

La aplicación permite a los usuarios rellenar, firmar y generar documentos en formato PDF relacionados con la LOPD y el consentimiento de los clientes. La interacción entre el cliente y el servidor se realiza mediante peticiones HTTP, especialmente a través de operaciones POST y GET.

En el lado del cliente, la aplicación utiliza tecnologías web estándar como HTML, CSS y JavaScript. Además, incorpora bibliotecas como SweetAlert para la creación de alertas, Signature Pad para la firma de documentos y html2canvas para la generación de documentos en formato PDF. La interfaz de usuario es responsiva, adaptándose tanto a dispositivos móviles como a tablets.

En resumen, este proyecto ofrece una solución completa para la gestión eficiente de la documentación de LOPD y consentimiento de los clientes, facilitando la comunicación entre el personal médico y los pacientes a través de una interfaz amigable y funcional.

Automatización de Creación de Cuentas de Correo Electrónico

En mi serie de proyectos centrados en la automatización con Python y la integración de API, destaca una solución especialmente eficaz: la optimización de creación de cuentas de correo electrónico. Este script fusiona diversas tecnologías y bibliotecas, representando un hito en la eficiencia y la simplicidad de la gestión de datos.

Uno de los componentes fundamentales de este proyecto es su capacidad para trabajar con un archivo de texto, el cual contiene información clave, como nombres, apellidos y direcciones de correo electrónico. La magia comienza con el uso de expresiones regulares para extraer y organizar estos datos en un diccionario fácilmente manejable.

La seguridad es una prioridad, y es por eso que el script implementa una función especializada para generar contraseñas aleatorias. Estas contraseñas cumplen con criterios específicos, excluyendo caracteres no deseados y asegurando la presencia de al menos un símbolo especial.

La gestión segura de datos es fundamental, y aquí es donde entra en juego la biblioteca pykeepass. El script se conecta con una base de datos de KeePass, proporcionando funciones para agregar nuevas entradas con verificación de duplicados y guardando los cambios de manera efectiva.

La función recuestFunc realiza solicitudes HTTP a la API de Dinahosting, utilizando credenciales y parámetros específicos para generar cuentas de correo electrónico de forma automática.

Este proyecto no solo automatiza tareas tediosas, sino que también mejora la eficiencia y seguridad en la creación y gestión de cuentas de correo electrónico.
